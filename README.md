# 🦋 LC-Executive-Dashboard

**LifeCharter Executive Dashboard** - An AI-powered command center for AmiLynne "Babs" Carroll to manage multiple businesses from a single unified interface.

## Vision

"One dashboard. All businesses. Complete clarity. Effortless execution."

## Businesses Supported

1. **LifeCharter** - Core transformation business
2. **LifeCharter Command Suite** - Business infrastructure
3. **AmiLynne Speaks** - Speaking & training
4. **Business in a Bot** - AI consulting
5. **Carroll Media** - Media production

### 🌅 TODAY (Executive Morning Brief)
- AI-generated daily summary

- Priority tasks

- Social media schedule
- Quick action bar

- Unified calendar (iCloud, Gmail, custom domain)
- Color-coded by business
- Smart conflict detection
- Meeting intelligence
- Optimal scheduling

### 📧 INBOX (Email Management)
- Unified inbox for all accounts
- AI-powered prioritization
- Smart categories
- Draft generation in Babs' voice
- Template library

- Multi-business P&L tracking
- Income & expense monitoring
- Bill calendar (week/month/quarter)
- Bank integration via Plaid
- Cash flow projections

- Kanban board with drag-and-drop

- Customizable columns

- AI smart suggestions

- Content creation studio

- Universal inbox for comments/DMs

### 🎓 LEARN (Training Video Library)
- Auto-import from Zoom & Read.ai

## Tech Stack

- **Frontend:** Next.js 14, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** Node.js, Next.js API Routes, tRPC
- **Database:** PostgreSQL, Redis
- **AI:** OpenAI GPT-4, Anthropic Claude, Whisper API
- **Infrastructure:** Vercel, Supabase

## Getting Started

```bash
# Clone the repository
git clone https://github.com/amilynne-carroll/LC-Executive-Dashboard.git

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local

# Run development server
npm run dev
```
